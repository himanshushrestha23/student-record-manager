# Student Record Manager

## Description

A simple command-line application built in Python to manage student records. The application allows users to add, search, update, delete, and display student records through a menu-driven interface. Student records are stored using Python dictionaries.

## Features

- Add a new student and their marks.
- Search for a student by name.
- Update a student's marks.
- Delete a student record.
- Display all stored student records.
- Input validation for duplicate and missing students.

## Technologies Used

- Python 3
- Dictionaries
- Functions
- Loops
- Conditional statements

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run:

```bash
python student_record_manager.py
```
